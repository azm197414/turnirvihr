# Турнирная таблица клуба БПЛА "ВИХРЬ"

Интерактивная таблица для подсчета результатов зимнего турнира.

## Особенности
- Автоматический расчет баллов по формулам
- Ввод данных в реальном времени
- Личный и командный зачет
- Экспорт в CSV

## Использование
1. Откройте [страницу турнира](https://ваш-username.github.io/турнир-вихрь/)
2. Введите данные участников
3. Нажмите "Пересчитать все"
4. Используйте "Экспорт в CSV" для сохранения результатов

## Формулы расчетов
- **Баллы Литофф:** = (Угол/МАКС_Угол * 70) + (МИН_Время_ЛЗ/Время_ЛЗ * 30)
- **Баллы Квадросим:** = (МИН_Время_КЗ * 100) / Время_КЗ
- **Баллы Трассы:** = (Круги/МАКС_Круги * 100) - Штраф
- **Баллы ЛЗ:** = Баллы_Литофф + Баллы_Квадросим + Баллы_Трассы
- **Баллы КЗ:** = Сумма Баллов_ЛЗ обоих участников команды

## Защита от изменений
Таблица защищена от несанкционированных изменений:
- Запрещено редактирование HTML/CSS через консоль разработчика
- Запрещено копирование формул
- Данные сохраняются только через экспорт в CSV
